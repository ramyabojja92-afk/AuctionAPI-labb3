# Labb3 Fullstack Auction

This project is a fullstack auction website built with ASP.NET Core Web API and React.

## Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- REST API for auctions and bids

## Frontend
- React
- TypeScript
- Vite
- Fetch API for backend communication

## Features
- User registration and login
- Create auctions
- View auctions
- Place bids
- Auction details page
